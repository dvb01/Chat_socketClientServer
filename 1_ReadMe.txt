Мой проект чата. Делал его 2020г. 
Сейчас бы много делал по другому из-за полученого нового опыта разработки.
Проект не является законченым и полностью работоспособным.
Но вы можете запустить Demo в папке Win32 как демонстрацию выполняя шаги по запуску.
1)Запустить Win32/Demo/Server/SocketServer.exe
2)Запустить Win32/Demo/Client1/SocketClient.exe
3)Запустить Win32/Demo/Client2/SocketClient.exe
4)Зарегистрировать аккаунт Client1
5)Зарегистрировать аккаунт Client2
6) Нужно найти себе собеседника
   1) в поисковике аккаунта Client1 справо вписать логин  Client2
   2)Кликнуть по нему
   3) Можно начинать переписку.

в папку Win32 есть скрин интерфейса
все данные сохраняются локально связь с интернетом не требуется.
но при желании сервер можно разместить удаленно и прописать в клиентах где он находится.
В любом случаии это просто демо и не более.
Данные не шифруются.
Протокол передачи обычный tcp на основе Winapi.WinSock
